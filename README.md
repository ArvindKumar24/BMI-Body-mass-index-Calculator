# BMI Calculator

A responsive and user-friendly Body Mass Index (BMI) Calculator built using HTML, CSS, and JavaScript. This application allows users to calculate their BMI using either *Metric (kg/cm)* or *Imperial (lbs/in)* units.

## Features

- Supports both Metric and Imperial units
- Real-time BMI calculation
- Clear and colorful visual indicator for BMI range
- Simple and intuitive interface
- Responsive design for different screen sizes

## How to Use

1. Select the unit system:
   - *Metric (kg/cm)*
   - *Imperial (lbs/in)*
2. Enter your weight and height.
3. Click the *"Calculate BMI"* button.
4. View your BMI result and the health category it falls under.

## BMI Categories

- Underweight
- Normal weight
- Overweight
- Obesity

## Technologies Used

- HTML5
- CSS3
- JavaScript
